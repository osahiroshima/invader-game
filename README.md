# invader-game

A simple terminal-based space invader clone written in Python using the `curses` module. Use the arrow keys to move and the space bar to shoot the invaders.

## Requirements

- Python 3 (with the built-in `curses` module)

## Running the Game

Execute the script directly:

```bash
python3 invader_game.py
```

The game runs inside your terminal. Move the player left and right with the arrow keys and shoot by pressing the space bar. When all invaders are destroyed you win. If an invader reaches the bottom row, the game ends.
