# Invader Game

This repository contains a minimal shooting game implemented in Processing. The sketch is provided in `InvaderGame.pde` and can be opened with the Processing IDE.

## Running the Game
1. Open **Processing** (https://processing.org/).
2. Load the `InvaderGame.pde` file from this repository.
3. Click the **Run** button.
4. Move the ship with the **LEFT** and **RIGHT** arrow keys and press **SPACE** to shoot.
5. If all enemies are destroyed you win, if an enemy reaches the player line the game ends.

Press the **R** key to restart after the game ends.
